Quick-howto på exportpluginen:

Kopiera till pluginkatalogen (maya/bin/plugins).
Ladda in med plugin-managern.

Flaggor

* '-dir <katalognamn>' anger var resultatet skall skrivas
* '-sel' exporterar endast valda objekt

Animationsflaggor

* '-s <startframe>' start
* '-e <endframe>'   slut
* '-b <byframe>'    steg

Användningsexempel:

exportReaper -dir "c:\\mittprojekt\\modeller" -sel
